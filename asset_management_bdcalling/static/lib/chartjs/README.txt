Place chart.umd.min.js in this directory.

Download from:
  https://cdn.jsdelivr.net/npm/chart.js@4/dist/chart.umd.min.js

Then rename to: chart.umd.min.js

After placing the file, restart the Odoo server and regenerate assets:
  ./odoo-bin --dev=all
or
  Settings > Technical > User Interface > Update Assets
